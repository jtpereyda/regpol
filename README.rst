regpol
======

Read Registry.pol files on Linux (or anywhere Python runs!). Not to be confused with
`regpol <https://petri.com/regpol>`_.

Install
-------

Requires Python 3.7 or greater.

Using `pipx <https://github.com/pipxproject/pipx>`_:

::

    pipx install regpol


Example
-------

::

    > regpol Registry.pol
